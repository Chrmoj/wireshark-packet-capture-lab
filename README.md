# wireshark-packet-capture-lab

## Overview
This lab project utilizes the network packet anaylzer wireshark to utilize its filters to capture key protocols in Windows.
Capturing traffic to understand what is transpiring in the network.

## The types of traffic captured
- **DNS** - Captured through `nslookup wireshark.org`
- **ICMP** - Captured through `ping 8.8.8.8 testing connectivity to google's public DNS server`
- **TCP 3 way handshake** - Captured through using a tab and searching to github.com
- **TCP port traffic 80 & 443** - Captured HTTP and HTTPS requests

## Skills learned in project
- Learning to read packages and saving them
- Learning wireshark filters and applying them
- Applying key protocols to understand network data
